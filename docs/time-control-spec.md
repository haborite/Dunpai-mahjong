# 持ち時間仕様

## 設定値

- 標準の持ち時間: 5秒
- 予備の持ち時間: 10秒
- 表示形式: `標準 + 予備`
- 予備時間の上限: 10秒

## 消費と回復

- 各操作では、最初に標準時間を消費する。
- 標準時間を使い切った後だけ予備時間を消費する。
- 標準時間は操作のたびに5秒へ戻る。
- 予備時間は局の間、プレイヤーごとに持ち越す。
- 自分のツモ巡で1秒以内に操作すると、予備時間を1秒回復する。
- 立直後の自動ツモ切りも即時応答として1秒回復する。
- 鳴きへの応答および鳴いた直後の打牌では回復しない。
- 予備時間は各局の開始時に10秒へ初期化する。

## 時間切れ

- 自分の打牌時はツモ牌を自動で打牌する。
- 鳴き選択時は自動でパスする。
- 残り3秒から毎秒警告音を鳴らし、最後の1秒は強い警告音にする。
- 時間切れ判定と自動操作はサーバーが行う。クライアント表示は案内用途とする。

## 旧実装からの修正点

- 固定30秒の打牌時間と固定8秒の鳴き時間を廃止した。
- 予備時間の保持、消費、回復、局開始時の初期化を追加した。
- 打牌にもカウントダウン表示を追加した。
- 不正操作や処理エラーでタイマーが満額に戻らないよう、元の締切を維持する。
- 鳴き待ちは各プレイヤーの残り予備時間に応じて個別に締切を管理する。
